### How to run application

1. Setup .env file for server inside **server** directory

   ```
   PORT =
   DB_URL =
   JWT_SECRET =
   CLOUDINARY_CLOUD_NAME =
   CLOUDINARY_API_KEY =
   CLOUDINARY_API_SECRET =
   ```

2. Install dependencies

   ```sh
   cd client && npm i
   cd ..
   cd server && npm i
   cd ..
   ```

3. Start the server, then run client

   ```sh
   cd server && npm start
   cd ../client && npm run dev
   ```
